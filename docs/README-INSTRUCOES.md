# Instruções da Atividade

Este repositório faz parte da atividade prática da disciplina de **Documentação Técnica em DevOps**.

## Objetivo do grupo
Registrar uma decisão arquitetural relevante da GreenEnergy em formato ADR, com contexto, decisão, alternativas e consequências.

## Documentação que deve ser produzida
- Arquivo final: `docs/adr-001.md`
- Referência de estrutura: `docs/adr-template.md`

## Regras da simulação
- Esta atividade é uma simulação acadêmica.
- Não é necessário implementar software ou infraestrutura.
- O foco é qualidade de registro decisório e clareza de trade-offs.

## Forma de trabalho
- O trabalho deve ser realizado no fork do grupo.
- O grupo deve justificar a decisão com base no contexto apresentado.
